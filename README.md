##v1.0.0 of Chain Connection

###built using Node.js, React, Pixi.js and Socket.io

####to build run:
```

npm install

webpack-dev-server --progress --colors

```

####now the game is available on localhost:8080

####to run multiplayer:

```
./server

npm install

run node index.js || pm2 index.js || forever index.js

```
